## 🧠 Project Overview

This project builds and compares three regression models to predict red wine quality scores (3–8) from 11 chemical properties. It follows a complete ML workflow:

```
Load Data  →  EDA (dtale)  →  Preprocessing  →  Feature Engineering
    →  Model Training (Before Tuning)  →  Fine-Tuning (GridSearchCV)
        →  Model Comparison  →  Best Model Saved  →  Streamlit App
```

---

## 📊 Dataset

- **Source:** [UCI Machine Learning Repository — Wine Quality](https://archive.ics.uci.edu/ml/datasets/wine+quality)
- **File:** `winequality-red.csv`
- **Rows:** 1,599 samples
- **Target:** `quality` score (integer 3–8)

### Features

| Feature | Description |
|---|---|
| `fixed acidity` | Tartaric acid concentration |
| `volatile acidity` | Acetic acid — causes vinegar taste |
| `citric acid` | Adds freshness and flavor |
| `residual sugar` | Remaining sugar after fermentation |
| `chlorides` | Salt content |
| `free sulfur dioxide` | Prevents microbial growth & oxidation |
| `total sulfur dioxide` | Free + bound SO₂ |
| `density` | Depends on alcohol & sugar |
| `pH` | Acidity level (0 = very acidic) |
| `sulphates` | Antimicrobial / antioxidant additive |
| `alcohol` | Alcohol percentage (% vol) |



## ⚙️ Preprocessing Steps

1. **Duplicate removal** — drops repeated rows
2. **Outlier removal** — IQR × 3 filter per feature (conservative, preserves edge wines)
3. **Feature engineering** — three derived features added:

```python
df["acidity_ratio"]   = df["fixed acidity"]       / (df["volatile acidity"]      + 1e-9)
df["so2_ratio"]       = df["free sulfur dioxide"]  / (df["total sulfur dioxide"]  + 1e-9)
df["alcohol_density"] = df["alcohol"]              / df["density"]
```

4. **Train/Test split** — 80% train, 20% test (`random_state=42`)
5. **StandardScaler** — fitted on train, applied to both sets

## 🤖 Models & Fine-Tuning

### Random Forest Regressor

```python
# Before tuning
RandomForestRegressor(n_estimators=100, random_state=42)

# GridSearchCV param grid
{
    "n_estimators":      [100, 200, 300],
    "max_depth":         [None, 10, 20],
    "min_samples_split": [2, 5],
    "max_features":      ["sqrt", "log2"]
}
```

### Gradient Boosting Regressor

```python
# Before tuning
GradientBoostingRegressor(n_estimators=100, random_state=42)

# GridSearchCV param grid
{
    "n_estimators":  [100, 200, 300],
    "learning_rate": [0.05, 0.1, 0.2],
    "max_depth":     [3, 5, 7],
    "subsample":     [0.8, 1.0]
}
```

### XGBoost Regressor

```python
# Before tuning
XGBRegressor(n_estimators=100, random_state=42)

# GridSearchCV param grid
{
    "n_estimators":     [100, 200, 300],
    "learning_rate":    [0.05, 0.1, 0.2],
    "max_depth":        [3, 5, 7],
    "subsample":        [0.8, 1.0],
    "colsample_bytree": [0.8, 1.0]
}
```

All models use **5-fold cross-validation** and are evaluated on `R²`, `MAE`, and `RMSE`.

## 📦 Saved Artifacts
